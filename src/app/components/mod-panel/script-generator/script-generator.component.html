Generate Cleanup & Update Script from Results

This script will delete all the original files of the available mods, move all unavailable mods to an "outdated" folder and download all updated mods.

To execute this script you need to have either PowerShell (Windows) or Bash (Mac & Linux) installed on your system.
Copy the generated script below and run it in your terminal in the mods folder, by right-clicking in the folder and selecting "Open in Terminal" or similar.
